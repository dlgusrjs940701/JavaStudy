종류 : default, public, private

source
원시프로그램 : 사람이 작성 (.java)
기계어 번역(0,1의 조합) : 컴파일
목적프로그램 : 기계가 이해 (.class) 

언어의 종류 : 컴파일 vs 인터프리터
컴파일언어는 자바로, 인터프리터언어는 자바스크립트로 공부했다 

thread : 코드를 목적프로그램에 실행시켜주는 것
자바에서 실행할때 스레드 단위로 코드를 실행
